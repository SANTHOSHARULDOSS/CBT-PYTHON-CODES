# Cryptocurrency and Blockchain Lab

Python programs for Blockchain core concepts — Merkle Tree, Hash Generation, and Blockchain.

---

## Files

| File | Experiment |
|---|---|
| `merkle_tree.py` | Merkle Tree Generation |
| `universal_hash_generator.py` | Universal Hash Generator |
| `blockchain.py` | Block and Blockchain Generation |

---

## Installation

```bash
git clone https://github.com/SANTHOSHARULDOSS/Cryptocurrency-and-Blockchain-Lab.git
cd Cryptocurrency-and-Blockchain-Lab
pip install tqdm tabulate
```

---

## How to Run

```bash
python merkle_tree.py
python universal_hash_generator.py
python blockchain.py
```

## Experiment 1 — Merkle Tree

Build a Merkle Tree from transactions, verify with Merkle Proof, detect tampering.

```
[1]  Build Merkle Tree
[2]  Build with Sample
[3]  Verify a Transaction
[4]  Tamper Detection Demo
[5]  Display Current Tree
[0]  Exit
```

---

## Experiment 2 — Universal Hash Generator

Hash any text, file, folder, or existing hash. Supports 10 algorithms.

```
[1]  Hash a Text / String
[2]  Hash a File
[3]  Hash a Folder / Directory
[4]  Hash of a Hash
[5]  Compare ALL Algorithms
[6]  Change Algorithm
[0]  Exit
```

**Supported** : MD5, SHA-1, SHA-224, SHA-256, SHA-384, SHA-512, SHA3-256, SHA3-512, BLAKE2b, BLAKE2s
